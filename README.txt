2.8-inch screen using method: 

1.Copy MZ280H-PI-EXT.rar files to your raspberry pi SD card by a card reader.
2.Decompress MZ280H-PI-EXT.rar to the home directory. 
3.After decompress,enter MZ280H-PI-EXT/src directory,then enter the command "make" to compile. 
4.Copy "lcd" files to    /etc/init.d 
5.Enter  /etc/init.d  directory, enter the command "rc-update" 
6.Connect this screen to the P1 interface of your raspberry pi,then restart your PI.
